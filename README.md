# Airbnb Performance Dashboard (Power BI)

## Project Overview
This project presents an interactive Power BI dashboard analyzing Airbnb performance across listings, pricing, reviewer behavior, and city-level market share. The objective is to extract actionable insights to support data-driven business decisions.

## Objectives
- Analyze listing growth trends over time  
- Understand reviewer behavior and engagement  
- Evaluate trust indicators such as profile verification  
- Compare market share across cities  
- Examine pricing differences by property type  

## Dataset Description
The dataset includes:
- Listing details (ID, city, property type)
- Host information  
- Reviewer data  
- Price and room type  
- Review counts and ratings  

## Key Insights

### Listings Growth
- Rapid growth during 2012–2016, peaking around 2015–2016  
- Decline after 2016 with a brief recovery  
- Sharp drop during the COVID-19 period  

### Property Type Distribution
- Entire homes dominate listings  
- Private rooms show moderate growth  
- Shared and hotel rooms have minimal contribution  

### Reviewer Behavior
- Majority of users give only one review  
- Significant drop in repeat reviewers  
- Indicates low repeat engagement and reliance on new users  

### Trust Analysis
- 66.9% users have profile pictures and are verified  
- 32.6% have profile pictures but are not verified  
- Very few users lack profile images  
- Profile presence contributes to trust, but verification is inconsistent  

### Market Share by City
- Paris, New York, and Sydney contribute a major share of listings and reviews  
- Paris leads in both listings and engagement  

### Pricing Analysis
- Hotel rooms have the highest average price  
- Entire homes are moderately priced  
- Private rooms are the most affordable  

### Ratings Analysis
- Mexico City and Rio de Janeiro have the highest ratings  
- Hong Kong and Istanbul have comparatively lower ratings  

## Tools and Technologies
- Power BI (Data modeling, DAX, visualization)  
- Power Query (Data transformation)  
- Excel/CSV dataset  

## Data Modeling
- Implemented a star schema for efficient querying  
- Established relationships between listings, hosts, and reviews tables  
- Optimized performance by avoiding complex relationships
